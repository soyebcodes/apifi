APiFi is a modern, production-ready SaaS starter template built with Next.js 14, React 18, and Tailwind CSS. It comes with a landing page for lead capture, a protected admin dashboard, Clerk authentication, MongoDB integration, and rich UI components.

### Landing Page

- Hero section with headline, subheadline, and call-to-action.
- Features section highlighting the core product benefits.
- Testimonials section showcasing user feedback.
- Pricing section with 3 tiers.
- Email signup form storing leads in MongoDB.
- Dark / light mode toggle.
- Smooth animations with Framer Motion.
- Fully responsive and mobile-friendly design.

### Dashboard

- Protected routes using Clerk authentication.
- Sidebar navigation (Dashboard, Leads, Settings).
- Leads overview with total leads.
- Growth chart using Recharts.
- Dynamic leads table.
- Smooth page transitions and UI interactions.

### Tech Stack

- **Frontend:** Next.js 14, React 18, Tailwind CSS, shadcn/ui
- **Backend / API:** Next.js API routes
- **Authentication:** Clerk
- **Database:** MongoDB
- **Animations:** Framer Motion
- **Deployment:** Vercel
